# scripts

Thin Jupyter notebooks that drive the package in `../src/`. They
contain **no business logic** — only imports, calls and plots.

| Notebook               | What it shows                                                       |
|------------------------|---------------------------------------------------------------------|
| `toy_example.ipynb`    | 4-country demo, no external data needed                             |
| `real_data.ipynb`      | End-to-end FAOSTAT pipeline for nitrogen, with a production shock   |

FAOSTAT CSVs go in `../data/` (see `../data/README.md`).
